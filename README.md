# store-sales-data-analysis
This project focuses on analyzing retail store sales data to uncover business insights and support data-driven decision-making. The analysis explores sales performance, customer behavior, product trends, and revenue patterns using data analytics tools and visualization techniques.
Project Objectives
	•	Analyze overall sales performance (Revenue, Profit, Quantity Sold).
	•	Identify top-performing products and categories.
	•	Track monthly and yearly sales trends.
	•	Analyze regional performance and customer segments.
	•	Detect patterns that help improve business strategy and forecasting.

🛠 Tools & Technologies
	•	Microsoft Excel (Data Cleaning & Pivot Tables)
	•	SQL (Data Extraction & Queries)
	•	Power BI (Interactive Dashboard & Visualization)
	•	Python (Pandas, Matplotlib) – optional if used

📈 Key Insights
	•	Seasonal trends impacting sales performance.
	•	Best and worst performing product categories.
	•	Regional contribution to total revenue.
	•	Customer purchasing behavior analysis.

📂 Project Structure
	•	data/ → Raw and cleaned datasets
	•	sql/ → SQL queries used for analysis
	•	notebooks/ → Data analysis scripts (Python)
	•	dashboard/ → Power BI dashboard file
	•	images/ → Dashboard screenshots

🚀 Results

The project provides actionable insights that can help store managers optimize inventory, improve marketing strategies, and increase profitability.
